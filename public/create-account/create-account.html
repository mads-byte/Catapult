We seek to make financial literacy and wealth building knowledge accessible for all and level uneven playing fields. "One way in which financial literacy is gained is via intergenerational transfer from one’s parents and grandparents" according to a Massey University study on The Intergenerational Transfer of Financial Literacy. But what if financial literacy is lower in a community that has been historically marginalized? Catapult aims to start financial education early, preferably before youths enter the workforce. This way, they are fully equipped with the tools they need to be financially successful and start building wealth young.
